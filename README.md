# AS Cake Shop

Jednostavan one-page sajt napravljen u čistom `HTML`, `CSS` i `JavaScript`.

## Struktura

- `index.html` - glavni markup stranice
- `css/styles.css` - kompletan izgled i responzivnost
- `js/data.js` - podaci za cheesecake ponudu i galerije
- `js/translations.js` - srpski i engleski tekstovi
- `js/main.js` - interakcije, slideri, FAQ, jezik i kalkulator dostave
- `assets/images/` - ilustracije i vizuali

## Pokretanje

Dovoljno je da otvoriš `index.html` u browseru.

Za lokalni preview u VS Code možeš koristiti `Live Server`.

## Besplatan hosting

Najlakše opcije:

1. `GitHub Pages`
2. `Netlify`
3. `Cloudflare Pages`

Ako želiš potpuno besplatno i bez kupovine domena, dobićeš adresu tipa:

- `ascakeshop.github.io`
- `ascakeshop.netlify.app`
- `ascakeshop.pages.dev`

## Napomena

Kalkulator dostave koristi javne OpenStreetMap servise za geokodiranje i rutu, pa za rad moraš imati internet kada sajt bude otvoren u browseru.
